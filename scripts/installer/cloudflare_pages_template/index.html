RAG Bootstrap

This page provides a raw public SSH key and a copy of the post-install installer script so you can bootstrap new machines quickly.

Public SSH key

Public keys are available at /keys/rag_deploy.pub. Add this to your machine's ~/.ssh/authorized_keys.

ssh-ed25519 AAAAC3NzaC1lZDI1NTE5AAAAIBEXAMPLEKEYPLACEHOLDER user@example.com

One-line bootstrap

Run this as root on a new machine to fetch the installer and execute it. Replace the domain with your Pages domain.

curl -fsSL https://example.pages.dev/post_install.sh | sudo PUBLIC_KEY_URL=https://example.pages.dev/keys/rag_deploy.pub bash -s --

Installer script

The installer will be available at /post_install.sh. It installs docker, helm, kubectl and adds the public key to the target user.
